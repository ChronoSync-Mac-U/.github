## Overview

ChronoSync Advanced is a comprehensive file synchronization and backup application that handles bidirectional sync, one-way backup, and archiving between any combination of local drives, network volumes, and cloud storage with flexible rule management and scheduling.

Bidirectional synchronization keeps two locations perfectly matched in both directions, applying changes from either side to the other and resolving conflicts according to rules you define, making it practical to maintain mirrored working environments across devices.

Scheduled sync and backup tasks run automatically at configured intervals without manual initiation, maintaining consistent synchronization and backup currency without requiring you to remember to run them manually.

Archive mode preserves deleted and modified files in a versioned archive alongside the main sync destination, providing a change history that allows recovery of any previous file version from the backup without a full restore.

For IT administrators, developers, and professionals who need precise, reliable file synchronization and backup across complex storage environments, ChronoSync Advanced is the sync tool that handles the full range of scenarios with the rule flexibility and scheduling depth that professional sync workflows require.

## Performance

ChronoSync balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
